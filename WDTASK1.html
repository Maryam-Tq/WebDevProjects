History Of HTML

HTML is an abbreviation for hypertext markup language hypertext markup language
HTML tells the browser how to display content such as, links, images, text, headings, tables on webpages

HTML is like a skeleton of webpage
Think of HTML as - Noun, CSS as Adjective and JavaScript as Verb.

How does HTML work?

HTML uses "tags" to structure content. Each tag tells the browser what a piece of content is.

Structure of HTML

HTML has a structure that includes:

  1. Declaration: It specifies HTML version being used
  2. HTML root tag: It wraps all the content present
  3. Head section: Contains title , style, links and metadeta of HTML
  4. Body section: Contains the main content of HTML

History of HTML

HTML was created by Tim Berner Lee in 1991. It was designed to share and organize documents over the internet.

Timeline of HTML Versions

  • 1991 - HTML 1.0: The first version. Very basic, used for simple web documents.
  • 1995 - HTML 2.0: Introduced forms and tables. Supported basic web features.
  • 1997 - HTML 3.2: Added support for scripting languages like JavaScript.
  • 1999 - HTML 4.01: Improved support for multimedia and forms. Laid the foundation for modern websites.
  • 2014 - HTML5: The current version of HTML. Supports audio, video, graphics, and responsive design.
